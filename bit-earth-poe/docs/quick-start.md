# Quick Start Guide

## Prerequisites

Before you begin, ensure you have:

1. **Development Tools:**
   - Rust 1.70+ (`rustup --version`)
   - Node.js 18+ (`node --version`)
   - Docker & Docker Compose (`docker --version`)
   - Git (`git --version`)

2. **Blockchain Accounts:**
   - BitcoinOS and Bitcoin Testnet wallet
   - Charms SDK API key
   - Test tokens for gas fees

3. **Hardware (Optional for IoT):**
   - Raspberry Pi 4+ (for sensor reading)
   - Voltage/current sensors
   - Smart meter with data output
